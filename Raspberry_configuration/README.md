# Raspberry_configuration

Raspberry Pi setup files used for the mission hardware deployment.

## Contents
- `Pipeline.md`: setup pipeline and step-by-step installation notes.
- `config_files/`: OS/hardware config snippets and rule files.

## Usage
- Follow `Pipeline.md` to prepare Pi OS, enable UART, and install dependencies.
- Copy files from `config_files/` where needed (`config.txt`, `cmdline.txt`, etc.).
